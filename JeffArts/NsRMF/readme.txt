How to use=--------------------------[Break]Macro & UserCode[Break]----------------------------------------------------------------------------------------------[Break][Break]This tab will combine your skill and/or your inventory to single press and release it straight after you press a hotkey.[Break][Break]----------[Break]Macro[Break]----------------------------------------------------------------------------------------------[Break][Break]This is easy to use for combine, you can set all of your skill there[Break]the first Hotkey for trigger of all spell, checkbox on the right of first hotkey for 'point target skill', if skill you want to combine is point target check it. but if skill you want to combine is self-target type, don't need to check it.[Break][Break]e.g. Skill on Lina with her item Ethereal, Dagon.[Break]Lets say Ethereal placed on Numpad1 and Dagon on Numpad2. The rest is [T][D] and [G]. On first hotkey, type anything to trigger thoose spells, Like [A], then[Break][Break][A] [checked] [Num1] [T] [G] [D] [Num2][Break][Break]click Submit, when you press A (as a hotkey), your hero will automaticly spam her skill started by Ethereal and ended by Dagon just with single press.[Break][Break]don't forget to check activate box if you want activate macro or usercode.[Break][Break]---------------[Break]UserCode[Break]----------------------------------------------------------------------------------------------[Break][Break]Here you can set your own Spell even more than 5 spells and more than 4 combo with code;[Break][Break]Hotkey=Key1+Key2+Key3 etc...[Break][Break] e.g on Lina again[Break][Break]A=Numpad1+CLICK+T+CLICK+G+CLICK+D+CLICK[Break]Use "Click" (without quotes) if skill/item is a point target type[Break]press save, then submit.[Break][Break]On usercode, you don't need to check point target because its all code for point-target skill. But if your skill is self-target, use Macro without checking the checkbox.[Break][Break]----------------[Break]Self-Target[Break]----------------------------------------------------------------------------------------------[Break][Break]e.g. Sniper (Macro)[Break]Say it sniper need a soul ring on numpad7 before he use his ultimate, lets put his skill like this[Break][Break][T][unchecked][Num7][T][Break][Break]Everytime you release your ultimate, soul ring will automaticly pressed.[Break][Break]e.g. Invoker (UserCode)[Break][Break]Q=Q 3[Break]F=Q+E 2+R+F[Break]E=E 3[Break][Break]When you press 'Q' Invoker will release triple quas in time, When you press 'F' Invoker will release Forge Spirit Skill and so on.[Break]Don't forget to use a Space between Key and Number if you want to release same skill in time (Q 3, not Q3).[Break]----------------------------------------------------------------------------------------------
